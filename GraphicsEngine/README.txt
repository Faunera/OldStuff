Graphics Engine project for school


controls

W - Move forward along z axis
S - Move backwards along z axis
D - Move forward along x axis
A - Move backwards along x axis
SPACEBAR - Move forward along z axis
LEFT CTRL - Move backwards along z axis
Q - Reset back to starting position

Hold LEFT SHIFT - Make movement faster

E - Add teapot
L - Move light position
I -Change light color (changes based on camera position)
O - Change object color (changes based on camera position)
Numbad + - change texture
B - blinn lighting on
N - blinn lighting off
